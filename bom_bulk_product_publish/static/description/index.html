BOM Bulk Product Publish

Publish all unpublished products with one click

This module allows users to publish all unpublished products with a single click, saving time and effort when managing your product catalog in Odoo.

Key Features

  • One-click publishing of all unpublished products
  • Easy access from product list view
  • Success notification with the number of products published
  • Warning notification if all products are already published

How to Use

After installing the module:

  1. Go to the Products list view
  2. Click the "Publish All Unpublished Products" button in the list header
  3. All unpublished products will be published immediately
  4. A success notification will show the number of products published
